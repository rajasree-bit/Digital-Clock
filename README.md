# Digital Clock

A simple digital clock created using HTML, CSS, and JavaScript.

## About the Project
This project displays the current system time and updates every second using JavaScript.  
It uses the Date object and setInterval() for real-time clock functionality.  
A 7-segment digital font is used to give the clock an electronic display look.

## Technologies Used
HTML  
CSS  
JavaScript  

## Features
- Displays current time in HH:MM:SS format  
- Updates automatically every second  
- Uses 7-segment digital font  
- Simple and clean UI  

## Project Structure
index.html  
style.css  
script.js  
fonts/digital-7.ttf  

## How to Run
Open index.html in any modern web browser.
